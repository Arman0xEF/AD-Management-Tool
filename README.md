# AD-Management-Tool

A Python tool to manage Active Directory users, groups, and computers via PowerShell.

## Features

- Get domain information
- Check Domain Admin status
- Manage users (create, disable, enable, reset password)
- Manage groups (create, add/remove members)
- View statistics (users, groups, computers)

## Requirements

- Windows OS
- Python 3.x
- Active Directory PowerShell module
- Domain Admin privileges (for some operations)

## Usage
```bash
python ad_manager.py
```

## Menu Options

1. Get domain information
2. Get current user
3. Check if I am Domain Admin
4. Get user details
5. Check if user is in group
6. List custom groups
7. Count custom users
8. Count computers
9. Create new user
10. Disable user
11. Enable user
12. Reset user password
13. Create new group
14. Add user to group
15. Remove user from group
